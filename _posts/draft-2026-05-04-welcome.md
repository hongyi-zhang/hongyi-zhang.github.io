This is a starter blog post. Use it as a template for short notes, essays, project updates, reading lists, or links.

To add a new post, create a Markdown file in `_posts/` named like `YYYY-MM-DD-title.md` with front matter:

```yaml
---
layout: post
title: "Your Post Title"
excerpt: "A one-sentence preview."
---
```

Then write the post content below the front matter.

<div data-lang="cn" hidden markdown="1">

这是一篇示例博客文章。你可以把它作为短笔记、随笔、项目动态、阅读清单或链接整理的模板。

要添加新文章，请在 `_posts/` 中创建一个形如 `YYYY-MM-DD-title.md` 的 Markdown 文件，并添加如下 front matter：

```yaml
---
layout: post
title: "Your Post Title"
title_cn: "中文标题"
excerpt: "A one-sentence preview."
excerpt_cn: "一句中文摘要。"
---
```

然后在 front matter 之后写正文。
